# CHALLENGE HOY TRABAJAS

### VERSIONS USED

- ruby 3.3.0
- postgresql 13

### ENV VARIABLES

```
DATABASE_USERNAME=
DATABASE_PASSWORD=
DATABASE_PORT=
DATABASE_HOST=
```

### INSTALLATION

```sh
$ bundle install
$ bundle exec rails db:prepare
```

### RUN SERVER
```sh
$ bundle exec rails server
```
